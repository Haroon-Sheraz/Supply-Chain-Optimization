# Supply-Chain-Optimization
Supply Chain Optimization Project for a Beverage Company.
•	Initiated the project with an existing model showcasing a -13% ROI.
•	Conducted comprehensive analysis involving plant production, DCs, distances, customer demands, and locations.
•	Considered retailer demand data and linehaul transportation costs for inbound and outbound shipments.
•	Analyzed plant capacities, operational costs, variable costs, and distribution center characteristics.
•	Developed a model with the help of an Excel data solver.
•	Utilized Excel data solver to optimize the selection of plant locations and DCs for transshipments.
•	Achieved a remarkable 27% ROI improvement by incorporating linking constraints into the model
